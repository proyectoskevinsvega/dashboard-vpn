# UI Components (`src/components/ui`)

Esta carpeta contiene componentes atómicos altamente reutilizables.

## Guía de Uso

- **Puros**: No deben consumir datos de APIs ni contener lógica de negocio compleja.
- **Variantes**: Utiliza props para definir variantes visuales (ej. `variant="primary"`).
- **Accesibilidad**: Asegúrate de que todos los componentes sean accesibles (WAI-ARIA).
- **Reusable**: Si un componente se usa en más de una sección, debe estar aquí.
