# Secciones (`src/components/sections`)

Este directorio contiene los bloques de contenido principales de la Landing Page.

## Componentes

- `Hero.tsx`: Primera impresión con propuesta de valor y CTAs.
- `Features.tsx`: Rejilla bento detallando ventajas tecnológicas.
- `About.tsx`: Narrativa corporativa y métricas de confianza.
- `Pricing.tsx`: Tabla de planes y precios orientada a conversión.

## Reglas

- Deben ser secciones independientes (con su propio `<section id="...">`).
- Utilizan componentes de `ui` para botones y decoraciones.
- Implementan animaciones de entrada vía `framer-motion`.
